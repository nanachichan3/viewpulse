# Viewpulse

**Your YouTube watch history — visualized.**

Upload your Google Takeout watch-history.html and get instant analytics: top channels, watch time trends, peak hours, keyword clouds, binge days, and more. All data processing happens in your browser — nothing is sent to any server.

## Quick Start

```bash
pnpm install
pnpm dev          # → localhost:3013
```

## How It Works

1. Export your **Google Takeout** watch history (YouTube data → HTML format)
2. Upload the file on Viewpulse
3. Get a full analytics dashboard — all client-side, no backend

## Tech

Next.js 14 + TypeScript + Recharts (fully client-side parsing)
