## 1. 10 High-Intent Keywords

These keywords represent creators actively searching for a solution to their analytics problem. They are all either:
- Problem-aware searches ("YouTube Studio alternative")
- Solution-seeking searches ("YouTube analytics tool")
- Education/validation searches ("best YouTube analytics")

| # | Keyword | Intent Type | Difficulty | Search Volume Signal | Priority |
|---|---------|-------------|------------|----------------------|----------|
| 1 | YouTube analytics tool | Transactional/Commercial | Medium | High — explicitly searching for a tool | **P0** — primary target |
| 2 | YouTube analytics dashboard | Commercial | Medium | High — looking for a better UI | **P0** — product matching |
| 3 | best YouTube analytics | Commercial/Comparison | Medium-High | High — in decision phase | **P0** — comparison stage |
| 4 | how to see YouTube analytics | Informational | Low | High — huge volume, broad reach | **P1** — top-of-funnel |
| 5 | YouTube creator tools | Commercial | Medium | Medium — broad creator audience | **P1** — awareness + conversion |
| 6 | track YouTube views | Commercial | Low | Medium — wanting tracking functionality | **P1** — functional intent |
| 7 | YouTube performance tracker | Commercial | Low | Medium — data-driven creators | **P1** — power user intent |
| 8 | YouTube analytics for creators | Commercial | Medium | Medium — specific niche phrase | **P1** — specific ICP match |
| 9 | YouTube channel analytics | Commercial | High | High — competitive but high volume | **P2** — long-term target |
| 10 | YouTube Studio alternative | Commercial | Medium | High — pain-aware searchers | **P2** — pain-point anchor |

### Keyword Notes:
- **YouTube Studio alternative** (#10) is the highest-converting phrase because searchers have already been frustrated by Studio. These are warm leads.
- **how to see YouTube analytics** (#4) is informational but high-volume — great for blog post targeting, drives awareness traffic.
- Prioritize P0/P1 in paid search (if/when budget allows); P1/P2 in organic blog content strategy.

## 2. 5 Initial Blog Posts

## 3. On-Page SEO Recommendations (Landing Page)

### Meta Tags & Structure
| Element | Recommendation |
|----------|----------------|
| **Page Title** | "Viewpulse — YouTube Analytics Dashboard for Creators" (primary keyword first) |
| **Meta Description** | "See what YouTube Studio hides. Viewpulse gives creators the analytics edge — CTR, watch time, retention, and more. Connect YouTube in 30 seconds. Try free." (155 chars, includes CTA + keywords) |
| **H1** | One H1: "The YouTube Analytics Dashboard You've Been Waiting For" — or similar |
| **H2 Structure** | H2s should mirror the target keywords: "See Your Real CTR," "Track Watch Time by Video," "YouTube Studio Alternative" |
| **URL slug** | `/` (root — single page app) or `/youtube-analytics-dashboard` for landing variation |

### Keyword Placement (On-Page Density)
- **Primary keyword "YouTube analytics"** — include naturally in: H1, first paragraph, 2–3 H2s, CTA buttons, footer
- **Secondary keyword "YouTube Studio alternative"** — include in: a dedicated pain-point section, meta description, at least one H2
- **"YouTube creator tools"** — include in: features section, one H2
- Avoid keyword stuffing. Natural density of 1–2% is sufficient.
- Use semantic variations: "YouTube analytics dashboard," "YouTube creator analytics," "channel analytics"

### Content Sections (Landing Page)
1. **Hero** — H1 + subheadline + CTA ("Connect YouTube Free") + dashboard screenshot
2. **Pain Point** — "What YouTube Studio Doesn't Show You" — mirrors blog post 1's angle
3. **Features Grid** — CTR, watch time, retention, traffic sources — each with a one-liner + icon
4. **How It Works** — 3-step flow: Connect YouTube → See Data → Make Better Videos
5. **Social Proof** — (placeholder for testimonials/beta user quotes once available)
6. **Pricing** — 3 tiers, monthly/annual toggle
7. **FAQ** — Target informational keywords: "how to see YouTube analytics," "best YouTube analytics"
8. **CTA Footer** — "Start free. No credit card required."

### Technical SEO
| Item | Status |
|------|--------|
| SSL (HTTPS) | Required (Vercel provides) |
| Page speed | Next.js + Vercel = fast by default; optimize images, lazy load below-fold |
| Mobile responsiveness | Tailwind = mobile-first; test on real devices |
| Core Web Vitals | LCP, FID, CLS — all should be green on Vercel deployment |
| Structured data (JSON-LD) | Add Organization schema + Product schema for Viewpulse (pricing tiers as offers) |
| XML sitemap | Auto-generated by Next.js sitemap.ts |
| Canonical URL | Set to viewpulse.rachkovan.com |
| Open Graph tags | Include og:title, og:description, og:image (dashboard screenshot) for social sharing |

### Image/Visual SEO
- **Dashboard screenshot** (OG image + hero): Label it clearly in the filename `viewpulse-dashboard-youtube-analytics.png`
- **Alt text** on all images: descriptive, keyword-inclusive — e.g., "Viewpulse YouTube analytics dashboard showing CTR and watch time metrics"
- **Favicon**: Simple brand mark that works at 16×16

### Internal Linking Strategy
- Link from blog posts TO the landing page with anchor text like "try Viewpulse free" and "Viewpulse dashboard"
- Link from landing page FAQ section TO individual blog posts for depth
- As content grows: hub page (YouTube Analytics Guide) linking to all 5 blog posts + landing page

### Blog Subdirectory
- Host blog at `viewpulse.rachkovan.com/blog/` — subdomain or subdirectory both work; subdirectory passes more domain authority to main site.
- Each post needs: unique meta title, meta description, canonical URL, OG tags, image alt text.
- Post schedule: Launch with 1–2 posts, add 1 per week for first month.
