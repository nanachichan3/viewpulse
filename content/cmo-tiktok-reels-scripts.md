## Production Notes

**Visual style:** Dark mode dashboard — dark background, teal/red accents. Each script should show the actual Viewpulse UI briefly during the explanation.

**Text overlays:** Big, bold numbers and keywords. White text on dark backgrounds.

**Optimal posting times (TikTok):**
- 6–9 AM EST (morning scroll)
- 12–2 PM EST (lunch break)
- 7–9 PM EST (evening unwind)

**Hashtag strategy:** Always include #Viewpulse + 2–3 niche-specific tags. Avoid oversaturated tags like #youtube alone.

**A/B test:** Scripts 1 and 4 are strongest hooks. Post those first, track engagement, iterate.
